## Featured Research Projects

**Temporally-Consistent XAI Framework for Financial Fraud Detection**
Designed a multi-model XAI framework integrating LSTM for temporal pattern learning, XGBoost for static feature importance, and Transformer-based self-attention for global transaction context. Applied SHAP and LIME to generate temporally-stable explanation outputs across sequential fraud detection models.
`Python` `LSTM` `XGBoost` `Transformer` `SHAP` `LIME`

**Credit Card Fraud Detection Under Extreme Class Imbalance**
Addressed fraud detection in a dataset of 6.3M+ records where fraud represented only 0.0003% of observations. Engineered 9 domain-driven features and benchmarked seven model configurations across class-weighting and SMOTE strategies.
`Python` `XGBoost` `Random Forest` `SMOTE` `scikit-learn`

**Volatility Modeling and Tail Risk Estimation in Equity Markets**
Applied GARCH(1,1) to model time-varying conditional variance and capture volatility clustering in Netflix stock returns. Estimated Value at Risk under parametric and historical simulation approaches. Augmented with Bollinger Band regime detection.
`Python` `GARCH` `VaR` `Bollinger Bands` `Time Series`
